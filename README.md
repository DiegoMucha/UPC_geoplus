# TP_1ACC0218_3230_Geom
> Un lenguaje diseñado para facilitar la enseñanza y el aprendizaje de la geometría mediante instrucciones simples, representación visual interactiva y manipulación dinámica de figuras.
 
---
## Problemática y Motivación
 
La enseñanza de la geometría suele ser **muy teórica y poco interactiva**, lo que dificulta que muchos estudiantes comprendan cómo se construyen, transforman y relacionan las figuras geométricas.
 
Además, los profesores deben invertir tiempo en realizar gráficos y demostraciones manualmente —ya sea en pizarra o utilizando herramientas complejas que no están pensadas específicamente para la enseñanza educativa—. Esto hace que procesos como representar figuras, aplicar transformaciones o explicar propiedades geométricas resulten más largos y menos dinámicos.
 
**GeoLang** propone un lenguaje de programación orientado a la geometría que permite crear y manipular figuras mediante instrucciones simples y fáciles de entender, con los siguientes beneficios:
 
- Los **estudiantes** pueden experimentar con las figuras y comprender mejor los conceptos de forma práctica.
- Los **docentes** cuentan con una herramienta más rápida y didáctica para explicar los temas.

## Objetivos
 
Para desarrollar un lenguaje que implemente instrucciones geométricas, se plantean los siguientes objetivos:
 
- **Diseñar una gramática en ANTLR4** clara, ordenada y que siga las convenciones de nomenclatura para las reglas léxicas y sintácticas.
- **Desarrollar el lenguaje de programación** aplicando Python y la librería de ANTLR, controlando el flujo de ejecución mediante el patrón **Visitor**.
- **Facilitar la representación y manipulación** de figuras geométricas mediante un conjunto de instrucciones simples.
- **Visualizar las instrucciones procesadas** mediante una interfaz basada en HTML utilizando el elemento `<canvas>`.
